GYM WORKOUTS -PLANS

CHEST AND BACK

1. Bench press - 4 sets x 8-10 repetitions
2. Chins (or Lat pulldowns) - 3 sets x 10-12 repetitions
3. Dumbbell row - 3 sets x 10-12 repetitions
4. Dumbbell press - 3 sets x 12-15 repetitions
5. Face Pulls - 3 sets x 12-15 repetitions

LOWER BOADY
(LEGS AND CORE)

1. Bench press - 4 sets x 8-10 repetitions
1. Squats - 4 sets x 8-10 repetitions
2. Deadlift - 4 sets x 8-10 repetitions
3. Lunges - 3 sets x 12-15 repetitions per leg
4. Lying leg press on machine - 3 sets x 12-15 repetitions
5. Russian Twists - 3 sets x 15-20 repetitions

Arms and shoulders

1. Bicep Curls - 3 sets x 12-15 repetitions
2. Tricep Bench Dips - 3 sets x 12-15 reps
3. Lateral lift with dumbbells - 3 sets x 12-15 repetitions
4. Arnold Press - 3 sets x 10-12 repetitions
5. Hammer Curls - 3 sets x 12-15 repetitions
